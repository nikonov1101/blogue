Софт на моём макбуке

14 октября 2018 г. 17:22

Заметил, что набор софта на моём маке не менялся уже достаточно давно. Новые утилиты не приживаются, - попросту не вписываются в устоявшиеся work-flow.

Я уже писал сколько стоит спираченный софт на моём маке, теперь решил описать вообще весь набор.

Браузеры

Google Chrome - основной, с кучей полезных закладок, заботливо разложенных по директориям. Переключен на старый “квадратный” дизайн.
Плагины:
- AdBlock Plus и Disconnect - вырезают рекламу и сервисы отслеживания.
- Grammarly - встраивается в textarea и проверяет грамматику английского языка. Помогает с артиклями, is/are и прочим таким.
- SourceGraph - навигация по коду на гитхабе прямо из браузера, очень помогает делать код-ревью.

Opera - пользуюсь ради встроенного VPN чтобы ходить на всякие торренты.

Разработка

GoLang и PyCharm. Гораздо больше времени провожу в Goland т.е текущий проект пишем на Go. Плагины из стандартного репозитория, и все для удобства работы с вспомогательными языками: Makefile, Bash, Systemd Units, Protobuf.

Sublime Text - используется как временное хранилище небольших кусочков текста, правки длинных команд для терминала и редактирования небольших проектов где можно обойтись без всех возможностей полноценной IDE.
Плагины:
- Подсветка синтаксиса для: Dockerfile, Markdown, Protobuf, TOML, Ansible.
- JSON, XML и HTML beautifier/compressor.
- GitGutter - интеграция с гитом.
- SublimeLinter для Python и Golang.
- Интеграция с Github Gist, с возможностью одним хоткеем отправить текущий файл в гист и получить ссылку на него в буфере обмена.

iTerm2 - божественный эмулятор терминала, доступен по хотекю из любого места системы. По статистике RescueTime провожу в нём больше всего времени. Фича, которую хочу отметить особо - деление экрана на части с удобным переключением. Про жизнь в терминале обязательно напишу отдельно.

Утилиты

Почта - Mail.app + GPG Suite.

1password - хранилище паролей, кредиток и прочих там важных штук. Синхронизируется через iCloud с телефоном.

Тексты - iA Writer для написания больших текстов в блог. Встроенные Notes - для хранения небольших заметок, кусочков команд и выдержек из книг.

TODO - Wunderlist, имеет удобный хоткей для быстрого добавления заметок, понимает человекопонятную запись даты и сам назначает уведомление, умеет в теги. Веду в нем три листа: Inbox: всякая помойка, лишь бы выгрузить из головы - напиши статью в блог, заплати на хату, поресёрчи ХХХ <ссылка>. Покупки - кинул в корзину в ашане - вычеркнул. Рабочее: мелочи, рефакторинг, фичи которые не стоит оформлять в полноценные таски.

MindNode - рисовалка MindMap-ов, очень их люблю для структурирования каши в голове.

KeepingYouAwake - маленькая утилита, не позволяющая маку уснуть. Переключается одним кликом в трее. Постоянно включаю на работе, когда ноутбук подключен к зарядке - не рвутся SSH соединения, можно оставлять загрузку больших файлов и идти курить. Работает когда ноутбук заблокирован.

Monosnap - скриншоты и небольшой редактор для них. Умеет заливать в своё облако и сохранять прямую ссылку в буфер обмена. Использую два режима - скриншот области с немедленной загрузкой в облако, и засунь-эту-область-в-редактор.

RescueTime - трекалка времени за компом. Умеет считать продуктивное время (писал код) и бездарно просранное (смотрел ютуб). Использую исключительно для поддержания самодисциплины, никому отчеты не высылаю.

Паранойа

FireWall - встроенный в macOS, с запретом всех входящих и с включенным стелс-режимом.

VPN - Использую встроенный клиент для L2TP и TunnelBlick для OpenVPN.

DNS - DNSCrypt для шифрованного днс, иногда меняю сервера.
