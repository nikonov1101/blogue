Понадобилось размножать тестовые окружения. При этом данные требовались те же самые, что используются при разработке.

Создаем новую базу, используя текущую как шаблон:

CREATE DATABASE application_test WITH OWNER developer TEMPLATE application_dev;

Либо из командной строки:

createdb -O developer -T application_dev application_test
