GNU Coreutils на macOS

26 февраля 2017 г. 9:23

Очень мне необходимы “нормальные” find, sed и awk от GNU. Хоты бы для того, чтоб не запоминать отличия этих утилит от их BSD-аналогов, которые есть в макоси.

Установить всё и сразу из coreutils при помощи brew:

brew install coreutils

Так-же можно поставить каждую утилиту по отдельности:

brew install binutils
brew install diffutils
brew install ed --with-default-names
brew install findutils --with-default-names
brew install gawk
brew install gnu-indent --with-default-names
brew install gnu-sed --with-default-names
brew install gnu-tar --with-default-names
brew install gnu-which --with-default-names
brew install gnutls
brew install grep --with-default-names
brew install gzip
brew install screen
brew install watch
brew install wdiff --with-gettext
brew install wget

Флаг --with-default-names установит утилиты со стандартными именами без префикса g, тем самым заменит утилиты, поставляемые с macOS.

Добавить утилиты в PATH можно так:

export PATH="$(brew --prefix coreutils)/libexec/gnubin:/usr/local/bin:$PATH"
