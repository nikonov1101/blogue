python, SSL и говнище.

26 января 2017 г. 20:27

При тестировании интеграции с очередной платёжкой в моём пистоне на macOS отвалился кусок SSL. А после обновления пистона отвалился еще и virtualenv.

Пришлось чинить при помощи изоленты и homebrew:

brew install python3  # установит пистон3.6
brew link --overwrite python3  # создаст симлинки на новый пистон
hash -r  # обновит все, что шелль знает о штуках в PATH

Проверяем, что пистону виден новый SSL:

python3.6
>>> import ssl; print(ssl.OPENSSL_VERSION)
OpenSSL 1.0.2j  26 Sep 2016

Ставим virtualenv + wrapper для python3.6

python3.6 -m pip install virtualenv
python3.6 -m pip install --user virtualenvwrapper

Пересоздаём окружение для проекта, накатываем зависимости, проверяем работу с SSL.
