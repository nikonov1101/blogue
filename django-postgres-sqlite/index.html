Миграция Django-приложения с PostgreSQL на SQLite

12 августа 2018 г. 22:13

Когда-то я уже писал про перетаскивание Django со SQLite на PostgreSQL.

Сегодня мне понадобилось сделать наоборот и поменять базу на SQLite.

Сначала нужно экспортировать все данные всех таблиц:

./manage.py dumpdata > data.json

Меняем в приложении настройки базы для использования SQLite.
Затем нужно восстановить в пустой новой базе структуру таблиц, для этого отлично подойдут миграции:

./manage.py migrate

Затем разворачиваем данные приложения в пустую базу:

./manage.py loaddata data.json

Стартуем сервер как обычно, проверяем что всё на месте.
